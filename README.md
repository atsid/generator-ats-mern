# generator-ats-node-webapp
> A [Yeoman](http://yeoman.io) generator for generating NodeJS Web Applications 

## Getting Started

### Usage
```bash
> npm install yeoman -g
> npm install generator-ats-node-webapp -g
> yo ats-node-webapp
```
