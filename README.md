# generator-ats-expressjs
> A [Yeoman](http://yeoman.io) generator for generating MERN-stack seed projects (Mongo, Express, React, NodeJS)

## Getting Started

### Usage
```bash
> npm install yeoman -g
> npm install generator-ats-mern -g
> yo ats-mern
```
